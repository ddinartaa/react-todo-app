# Список задач (React)

Учебное приложение, созданное в рамках практической работы №4 по дисциплине "Верстка и создание приложений".

## Ссылка на проект

https://ddinartaa.github.io/react-todo-app

## О проекте

Приложение отображает список задач, которые загружаются из тестового API JSONPlaceholder. Есть две страницы: список задач и информация о приложении.

## Использованные технологии

- React
- Vite
- React Router
- GitHub Pages

## API

https://jsonplaceholder.typicode.com/todos
